## クイックスタート

### インストール

```bash
# npm
npm install -g {{PACKAGE_NAME}}

# yarn
yarn global add {{PACKAGE_NAME}}

# pnpm
pnpm add -g {{PACKAGE_NAME}}
```

### 基本コマンド

```bash
# ヘルプ表示
{{PACKAGE_NAME}} help

# プロジェクトセットアップ
{{PACKAGE_NAME}} setup

# ドキュメント一括生成
{{PACKAGE_NAME}} build
```
